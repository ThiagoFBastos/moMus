# moMus

site criado para estudar Node.js, Express e Sequelize 

## Requisitos

1. Node js
2. Mysql

## Uso

1. importar o arquivo moMus.sql para o mysql
2. inseirir no terminal o seguinte comando: node main.js
3. visitar http://localhost:8181/filme/ no navegador
